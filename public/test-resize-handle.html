Resize Handle Test

This page helps verify that the resize handle functionality is working correctly in the wireframe editor.

Test Steps:

  1. Open the wireframe editor in a new tab.
  2. Add a component (e.g., Container, Button, etc.) to the canvas.
  3. Click on the component to select it. You should see blue resize handles at the corners.
  4. Try dragging any of the resize handles to resize the component. The component should resize accordingly.
  5. Open the Properties panel on the right side and try changing the Width and Height values. The component should resize accordingly.
  6. Verify that both methods of resizing (handles and property inputs) work correctly.
